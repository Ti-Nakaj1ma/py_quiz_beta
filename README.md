〇概要  
このアプリはPythonに関する4択問題を出題するアプリです。難易度が4種類に分かれており、
print文などの基本的な文法からエラー文の意味を理解する問題などの難易度の高いものまで収録しています。
問題数は各100問を想定していますが、現在は50％程の問題数です。

〇操作方法  
スタートボタンを押して、難易度を選択します。その後、問題文が画面上部に出てくるので、正解だと思うボタンをクリックします。回答が終了すると正解数、正解率を表示し、間違った問題があれば「間違った問題」というボタンが出現します。それをクリックすることで間違えた問題の見直しを行うことが出来ます。「タイトルに戻る」ボタンを押すと最初の画面に戻ります。
